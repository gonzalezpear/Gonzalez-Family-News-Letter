# Images Folder

Place your newsletter images here.

## Recommended Structure

- `images/november-2025-featured.jpg` - Featured image for November 2025 newsletter
- `images/thanksgiving-2025.jpg` - Thanksgiving photos
- `images/december-2025-featured.jpg` - Featured image for December 2025 newsletter
- etc.

## Image Tips

- Use descriptive filenames (e.g., `november-2025-featured.jpg`)
- Recommended formats: JPG, PNG, WebP
- For featured images, aim for 16:9 aspect ratio (e.g., 1200x675px)
- For content images, any size works (they'll be responsive)
- Keep file sizes reasonable for web (under 500KB is ideal)
